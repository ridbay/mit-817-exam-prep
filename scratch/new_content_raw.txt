# BLOCK 4: THEORY QUESTIONS
1. Define Software Engineering and explain its importance
✅ Model Answer

Software engineering is an engineering discipline concerned with all aspects of software production, from specification to maintenance.

It is important because:

Software systems are complex and continuously evolving
The “software crisis” showed problems of cost, delays, and failures
It ensures systematic, disciplined, and measurable development

Example:
A banking system requires structured development to ensure security and reliability.

🔵 2. Explain the layered technology of software engineering
✅ Model Answer

Software engineering is a layered technology consisting of:

Quality focus – foundation ensuring high standards
Process layer – framework of activities (communication, planning, modeling, construction, deployment)
Methods layer – technical techniques (analysis, design, coding, testing)
Tools layer – automated support (IDEs, testing tools)

Example:
In a banking system:

Process defines steps
Methods define how to design
Tools support coding
🔵 3. Discuss the fundamental activities of software engineering
✅ Model Answer

The fundamental activities are:

Specification – defining system requirements
Development – designing and coding
Validation – testing correctness
Evolution – maintaining and updating

Example:
ATM system:

Specify withdrawal
Develop code
Validate functionality
Add features later
🔵 4. Explain Agile Software Development and its principles
✅ Model Answer

Agile is a flexible development approach emphasizing:

Working software over documentation
Customer collaboration over contracts
Responding to change over plans

Principles:

Incremental delivery
Continuous feedback
Simplicity

Example:
Food app developed in weekly increments based on user feedback.
